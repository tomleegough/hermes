Hermes Accounting is not yet advanced enough to use in a production environment. In lieu of a proper userguide or documentation, here is some guidance.

To get started:

  1. Add an organisation and select it
  2. Create some categories. Categories work like a simplified chart of accounts
  3. Create some bank accounts
  4. Add some transactions

Selecting a new organisation will re-orientate the whole Hermes application around that organisation. Categories do not align or sync across organisations.
