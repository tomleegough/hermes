Hermes Accounting was writting by me, Tom Lee-Gough, a CIMA qualified Accountant. I've been writing Hermes since September 2018. This is a long term goal of writing my own accounting software.

Hermes is written in Flask and Bootstrap, with themes from Bootswatch. It was written using PyCharm, a Python IDE. At the time of writing, Hermes is hosted on Digital Ocean

Hermes is free to use, and I offer no guarantees of any kind. Any use of Hermes is at the user's own risk. This may change at some point in the future, however, for now you cannot rely on this software. I cannot stress this enough. I will not accept ANY liability for use of the software.

It would be great to make Hermes open source. Some older code can be found on github (https://github.com/tomleegough/hermes). At the moment, I'm working on making the latest code available on github.
